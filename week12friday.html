Math

How do we add math?

(Medium)

Math

Import the Math.js from the CDN

Add below the other script tags

Make sure to close the script tags above!

Math

Add the Math config to the Reveal.initialize block at the end of the file


							Reveal.initialize({
								hash: true,
								slideNumber: 'h/v',
								math: {
								      mathjax: 'https://cdn.jsdelivr.net/gh/mathjax/mathjax@2.7.8/MathJax.js',
								      config: 'TeX-AMS_HTML-full',
								      TeX: { Macros: { RR: "{\\bf R}" } }
								    },

Note how this references another CDN to deliver something called mathjax (link)

Math

Add the RevealMath class to the list of plugins inside Reveal.initialize


							Reveal.initialize({
								hash: true,
								slideNumber: 'h/v',
								math: {
								      mathjax: 'https://cdn.jsdelivr.net/gh/mathjax/mathjax@2.7.8/MathJax.js',
								      config: 'TeX-AMS_HTML-full',
								      TeX: { Macros: { RR: "{\\bf R}" } }
								    },
								plugins: [ RevealHighlight, RevealMath ]
							});

Make sure this list of plugins matches the scripts you imported with the script tags!
